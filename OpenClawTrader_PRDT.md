# OpenClaw Trader - 产品需求文档 (PRD)

## 1. 产品概述

**产品名称：** OpenClaw Trader
**产品类型：** iOS 移动应用
**核心定位：** OpenClaw AI 平台的移动控制台 + 智能股票交易辅助工具
**目标用户：** OpenClaw 用户、量化交易者、投资顾问

---

## 2. 功能架构

### 2.1 消息模块

#### 2.1.1 通知中心
- OpenClaw 通知
  - Agent 执行完成通知
  - 工作流异常告警
  - Token 使用超限提醒
- 交易通知
  - 价格预警（自定义涨跌幅）
  - 交易机会提醒
  - 持仓异动提醒
  - 风险预警通知
- 消息分类筛选（全部/未读/Agent/工作流/交易）

---

### 2.2 交易模块

#### 2.2.1 持仓管理
- 持仓概览（总市值、今日收益、总收益）
- 持仓明细列表（搜索、排序：市值/收益/日涨跌）
- 持仓详情（持股数量、成本、当前价、盈亏分析）
- 持仓导入
  - 截图识别（持仓截图 OCR 解析）
  - 文本粘贴导入
  - 手动输入

#### 2.2.2 交易习惯分析
- 交易频率统计（日/周/月）
- 持仓周期分析（短线/中线/长线偏好）
- 交易风格识别：
  - 趋势跟踪者
  - 反转交易者
  - 价值投资者
  - 事件驱动者
  - 板块轮动者
- 盈亏比分析
- 止损止盈习惯评估
- 仓位管理风格（集中/分散、金字塔/倒金字塔）

#### 2.2.3 风险评估
- 仓位集中度风险
- 行业暴露度分析
- 波动率暴露评估
- 杠杆使用评估
- 综合风险评分（1-10分）

#### 2.2.4 AI 交易建议
- 基于 OpenClaw 分析的个性化建议
- 仓位调整建议
- 止损止盈点建议
- 交易时机提示
- 模拟回测验证建议有效性
- 建议优先级分类（高/中/低）

#### 2.2.5 收益报表
- 日/周/月/年收益统计
- 收益率曲线图
- vs. 大盘对比（沪深300/标普500等）
- 胜率统计
- 最大回撤统计
- 关键指标（夏普比率、盈利因子）

#### 2.2.6 历史交易
- 交易记录列表
- 买入/卖出筛选
- 交易详情查看

---

### 2.3 我的模块

#### 2.3.1 账户管理
- OpenClaw 连接
  - API 地址配置
  - API Key 配置
  - 连接状态显示
- 外观设置
  - 跟随系统
  - 日间模式
  - 深色模式

#### 2.3.2 Workspace 管理
- 多 Workspace 切换
- Workspace 创建
- Workspace 状态概览（活跃 Agent 数、运行中工作流、最近活动）

#### 2.3.3 模型管理
- 模型列表展示（名称、版本、状态）
- 模型切换与激活
- 模型参数配置（Temperature、Max Tokens、Top-P 等）
- Token 使用量统计与费用估算
- 模型性能监控（响应时间、错误率）

#### 2.3.4 Agent 管理
- Agent 列表与搜索
- Agent 创建向导
- Agent 配置（名称、描述、绑定的模型、系统提示词）
- Agent 状态监控（空闲/运行中/错误）
- Agent 对话历史查看
- Agent 快捷操作（启动/停止/重启/删除）

#### 2.3.5 工作流管理
- 工作流列表与分类（全部/活跃/暂停/错误）
- 工作流执行监控
  - 实时状态
  - 日志流
  - 步骤详情
- 工作流历史记录
- 触发器管理（定时、事件驱动）

---

## 3. UI/UX 设计方向

### 3.1 整体风格
- 深色主题为主（适合盯盘）
- 支持日/夜模式切换
- 金融/科技感设计语言
- 简洁高效的信息层级

### 3.2 底部导航结构
1. **消息** - 通知中心
2. **交易** - 持仓管理与交易辅助
3. **我的** - 账户与 OpenClaw 管理

### 3.3 核心页面
- 交易 Dashboard（持仓概览、AI建议快捷入口）
- 持仓分析页（交易画像、风险评估）
- 收益报表页（曲线图、胜率统计）
- OpenClaw Dashboard（Workspace概览、快捷操作）
- 模型列表 + 详情页
- Agent 列表 + 聊天页
- 工作流监控页（实时日志流）

---

## 4. 技术架构建议

### 4.1 前端
- **SwiftUI + UIKit 混合**
- **架构：** MVVM
- **网络：** URLSession / Alamofire
- **WebSocket：** 实时日志流
- **图表：** Charts（iOS 16+） / DGCharts

### 4.2 数据存储
- **SQLite.swift** 本地持久化
- **Keychain** 敏感信息（API Keys）

### 4.3 OpenClaw 接入
- REST API 对接
- WebSocket 实时通信
- OAuth 2.0 认证

### 4.4 OCR（持仓识别）
- Vision Framework（iOS 原生）
- 或云端 OCR API

---

## 5. 优先级规划

### Phase 1 (MVP)
- OpenClaw 连接与基础管理
- 持仓导入（截图 + 文本）
- 交易习惯基础分析

### Phase 2
- 风险评估
- AI 交易建议
- 价格预警通知
- 工作流监控
- 实时日志中心

### Phase 3
- 收益报表
- 历史交易管理
- 模拟交易
- 社区模块

---

## 6. 成功指标

- OpenClaw API 集成完成度
- 持仓识别准确率 > 90%
- 用户交易建议采纳率
- 用户留存率（周活/月活）
- App Store 评分目标：4.5+
